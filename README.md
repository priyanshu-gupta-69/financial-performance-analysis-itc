# Financial Performance & Ratio Analysis of ITC Ltd

## Key Insights
- ITC maintains consistently high profitability with strong net profit margins over the five-year period.
- Return on Equity (ROE) shows an improving trend, indicating efficient utilization of shareholders’ equity.
- Liquidity remains strong throughout the analyzed period, reflecting financial stability.
- Post-COVID recovery is clearly visible in both revenue and profit growth.

## Objective
The objective of this project is to analyze the financial performance of ITC Ltd over a five-year period (FY 2020–FY 2024) using financial ratio analysis and trend visualization.  
The analysis focuses on profitability, liquidity, and efficiency indicators using Python-based data analysis.

## Time Period
FY 2020 – FY 2024

## Key Metrics Analyzed
- Net Profit Margin  
- Return on Equity (ROE)  
- Current Ratio  
- Asset Turnover Ratio  

## Tools Used
- Python (Pandas, Matplotlib)
- Google Colab
- Excel (data preparation)

## Data Description
The dataset contains key financial indicators of ITC Ltd for FY 2020–FY 2024:
- `revenue`: Total revenue (₹ Cr)
- `net_profit`: Net profit (₹ Cr)
- `total_assets`: Total assets (₹ Cr)
- `equity`: Shareholders’ equity (₹ Cr)
- `current_assets`: Current assets (₹ Cr)
- `current_liabilities`: Current liabilities (₹ Cr)
